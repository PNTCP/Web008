Workshop: HTML Heading

Practice using <h1> – <h6> tags for themes with CSS.

1. Meaning of Headings (H1–H6)

HTML has six levels of headings.

  • H1 – The largest heading; used for the webpage title or main heading.
  • H2 – Subheading for dividing content sections
  • H3 – Subheading within H2
  • H4 – Drilling down another level
  • H5 – Minimal use, but emphasizes data sequencing.
  • H6 – Smallest; used for deep-level headings.

There should be only one H1 tag per page to aid SEO and document structure.

2. Examples of using H1–H6


        <h1>Welcome to My Website</h1>
        <h2>HTML Basics</h2>
        <h3>What is HTML?</h3>
        <h4>HTML Syntax</h4>
        <h5>Example Code</h5>
        <h6>Note: Case Sensitive</h6>

When displayed, it will appear in a tiered layout like this.

H1: Welcome to My Website

H2: HTML Basics

H3: What is HTML?

H4: HTML Syntax

H5: Example Code
H6: Note about heading size
